Formatos de Imágenes: JPG, PNG, WebP, AVIF, SVG

Publicado el 21 de enero de 2025

1. JPG/JPEG

El formato JPG es uno de los más populares debido a su excelente compresión y calidad de imagen para fotos. Se utiliza ampliamente en sitios web, redes sociales, y en fotografía.

Usos comunes:

  • Fotografías
  • Imágenes de alta calidad con poca necesidad de transparencia

2. PNG

El formato PNG es ideal para imágenes con transparencia. Este formato no pierde calidad al ser comprimido y es ampliamente utilizado para logotipos y gráficos web.

Usos comunes:

  • Imágenes con transparencia
  • Logotipos y gráficos con bordes nítidos

3. WebP

El formato WebP es moderno y ofrece una excelente compresión sin sacrificar calidad. Es compatible con la mayoría de los navegadores modernos.

Usos comunes:

  • Imágenes en la web para una carga más rápida

4. AVIF

AVIF es un formato de imagen que ofrece una excelente compresión y alta calidad, superando a WebP en algunas áreas. Aunque es relativamente nuevo, está ganando popularidad rápidamente.

5. SVG

SVG es un formato de imagen basado en vectores, ideal para gráficos y logotipos que necesitan mantenerse nítidos incluso cuando se escalas a tamaños más grandes.

Propiedades CSS para Imágenes

Publicado el 22 de enero de 2025

1. background-image y Propiedades Asociadas

La propiedad background-image permite establecer una imagen de fondo para un elemento HTML. También se pueden usar propiedades adicionales como background-position, background-size, y background-attachment para controlar cómo se muestra la imagen.

Esta es una imagen de fondo con propiedades CSS aplicadas.

2. object-fit y object-position

Las propiedades object-fit y object-position permiten controlar cómo una imagen se ajusta dentro de un contenedor <img>. La propiedad object-fit se puede usar para ajustar la imagen, por ejemplo, a cubrir el contenedor o adaptarse a su tamaño.

Filtros en CSS para Imágenes

Publicado el 23 de enero de 2025

1. Cambiar Brillo

Con el filtro brightness() puedes ajustar el brillo de una imagen, haciendo que se vea más clara o más oscura.

2. Cambiar Contraste

La propiedad contrast() ajusta el contraste de la imagen.

3. Filtros de Saturación

La propiedad saturate() permite cambiar la saturación de los colores en una imagen.

4. Desenfoque de Imagen

El filtro blur() aplica un desenfoque a la imagen.
