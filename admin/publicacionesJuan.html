Propiedades CSS para Imágenes

Publicado el 22 de enero de 2025

1. background-image y Propiedades Asociadas

La propiedad background-image permite establecer una imagen de fondo para un elemento HTML. También se pueden usar propiedades adicionales como background-position, background-size, y background-attachment para controlar cómo se muestra la imagen.

Esta es una imagen de fondo con propiedades CSS aplicadas.

2. object-fit y object-position

Las propiedades object-fit y object-position permiten controlar cómo una imagen se ajusta dentro de un contenedor <img>. La propiedad object-fit se puede usar para ajustar la imagen, por ejemplo, a cubrir el contenedor o adaptarse a su tamaño.

Filtros en CSS para Imágenes

Publicado el 23 de enero de 2025

1. Cambiar Brillo

Con el filtro brightness() puedes ajustar el brillo de una imagen, haciendo que se vea más clara o más oscura.

2. Cambiar Contraste

La propiedad contrast() ajusta el contraste de la imagen.

3. Filtros de Saturación

La propiedad saturate() permite cambiar la saturación de los colores en una imagen.

4. Desenfoque de Imagen

El filtro blur() aplica un desenfoque a la imagen.
